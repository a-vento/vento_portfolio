MINDFUL STEPS DESIGNS

A journey towards enhancing user engagement through design

Project Overview

The purpose of the Mindful Steps research study is to examine ways to increase physical activity in veterans with Chronic Obstructive Pulmonary Disease (COPD) and chronic musculoskeletal pain.

Solution

A comprehensive redesign was implemented to create a user-friendly desktop and mobile website.

Research Phase

1. Conducted stakeholder interviews

Key insights:

Preference against complex UI elements

Desire for a familiar website structure

Importance of clear visual and text hierarchies

User Needs Analysis

2. Developed personas and identified critical user needs

Top user requirements:

Minimized interaction for ease of use

Appealing visual design

Clear information presentation

Desktop Redesign

3. Iterated over desktop designs

Mobile Interface Design

4. Introduced a mobile-friendly interface

Design adjustments:

Adjusted grid system to account for smaller views

Ensured readability and hierarchy
